Introduction

Java language was developed by James A Gosling in association with Sun Microsystem

Java language is widely accepted as desktop application development technology.

Java programming is used to develop Internet applets or java applets.

An applet is a program written in java programming language that can be included in an html page.

When we use a java technology enabled browser to view a page that contains an applet, the applets code is transfered to our system and executed by browser's interpreter.

Basic Features

Some basic features of java are enlisted below:-

  1. It is an object oreinted programming language.

  2. Java programs are both compiled and interpreted.

  3. Java programs are platform independent.

  4. Java is a robust programming language.

  5. It is a multithreaded language.

  6. It can access data from a local system as well as from internet.

  7. Java is a case sensetive language.

  8. Java doesn't require any preprocessor(#) or inclusion of header files for creating a java application program.

Escape Sequence

'\' is known as the escape sequence of java.It is used for:-

  • To represent the non graphical characters inside a java program.

    • \n - new line
    • \t - tab
    • \b - backslash
    • \r - carriage return
    • \' - single quotes
    • \" - double quotes

  • To neglect the meaning of some java punctuators
    example:- , ; "" ''etc.

Tokens

The smalles meaningful unit of a java program is called a token.

Tokens are of five tyoes:-

  1. keywords
  2. identifiers
  3. literals
    • Integers
    • Real
    • Character
    • String
    • Boolean
    • Null
  4. Operators
  5. Punctuators

Keywords

These are java reserve words which convey some special meaning to the java compiler. Example:- if, else, for, break, etc.

Identifiers

Identifiers are the usere defined names to identify the different elements of a java program. Example:- class_name, variable_name, method_name, etc.

Rules of writting identifiers:

  1. Identifier should be unique in a program.
  2. They should not start with a digit.
  3. Upper case and lower case are treated differently.
  4. Keywords are not used as identifiers.
  5. Identifers can use all the alphabets(A-Z & a-z) and digits (0-9) and two special characters'_'(underscore) and '$'(dollar).

Literals

The fixed values or constants used in a program is called literals.
There are six types of literals.
  1. Integer

    The literals with no fraction part in them are called Integer literal.They can be both positive and negative.

  2. Real

    The number with any fraction part in them are called integer literal. They can be both positive and negative.

  3. Boolean

    The two reserve words of java 'true' and 'false' are called boolean literal.The words 'true' and 'false' are always written in small.

  4. Character

    Zero or one characters inclosed within single quotes '' are called as character literal.

  5. String

    Zero or more characters inclosed within double quotes "" are called as String literal.

  6. Null

    The java reserve word 'null' which is used to create an empty space is called a null literal.'\0'

Operators

The operators are the special symbols used in a java program to perform some action with the given value.

Punctuators

These are the special symbols used in a java program to seperate the different parts of a program.
